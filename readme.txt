== Installation ==

**Install**

1. Unzip the `sp-wavewatch-surf-widgets.zip` file. 
1. Upload the the `sp-wavewatch-surf-widgets` folder (not just the files in it!) to your `wp-contents/plugins` folder. If you're using FTP, use 'binary' mode.

**Activate**

1. In your WordPress administration, go to the Plugins page
1. Activate the Wavewatch Widget plugin
1. Go to the Design > Widget page and place the widget in your sidebar in the Design
1. In the widget, select either "conditions" or "map", the region and the size of your widget. 

If you find any bugs or have any ideas, please mail us.
